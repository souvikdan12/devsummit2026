# AI Resume Analyzer

A full-stack web application that bridges the gap between candidate resumes and industry requirements providing skill gap detection, career roadmaps, and data-driven insights to help professionals optimize their trajectory.

---

## Features

- **Skill Gap Analysis** — Identifies technical deficiencies and surfaces curated learning resources to close them
- **Strategic Roadmaps** — Generates short-term (1–3 month) and long-term career development plans
- **Corporate Alignment** — Scores your resume against specific organizations' requirements
- **Data Visualization** — Interactive SVG-based donut, bar, and radar charts for match scoring and skill distribution
- **Role Identification** — Recommends optimal job titles based on your extracted skill set
- **Cover Letter Generator** — Produces tailored cover letters from resume data
- **Skill Assessments** — Timed quizzes to validate and benchmark your proficiencies
- **Career Path Simulation** — Multi-year progression modeling based on current trajectory
- **Portfolio Enhancement** — Curated project ideas to strengthen specific weak areas

---

## Tech Stack

**Frontend**
- React 19 + Vite
- React Router v7
- Custom glassmorphism design system (light/dark mode)
- Zero-dependency SVG charts

**Backend**
- Node.js + Express 4
- Multer (file uploads)
- pdf-parse (text extraction)
- Advanced text analysis for resume parsing

---
# Build Commands
- npm install in both folders(frontend & backend)
- run "npm run dev" first in backend and then in frontend
